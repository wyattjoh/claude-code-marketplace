# Wyatt Johnson's Claude Code Marketplace

A collection of Claude Code plugins.

## Installation

Add this marketplace to Claude Code:

```shell
/plugin marketplace add wyattjoh/claude-code-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [deno-lsp](https://github.com/wyattjoh/deno-lsp-claude-plugin) | Deno Language Server integration for Claude Code |

## Installing Plugins

After adding the marketplace, install plugins with:

```shell
/plugin install deno-lsp@wyattjoh-marketplace
```

Or use the interactive plugin browser:

```shell
/plugin
```
